Design and implementation of an automated water management system for gravity-fed irrigation in Burkina Faso

The objective of this project is to design an automated irrigation system that optimizes water usage in agriculture. By using sensors and automation, the system provides water only when necessary, reducing waste, improving crop yields, and supporting sustainable farming practices.

 

 

 
  • The system is controlled by an Arduino board, automatically determines when and how much to irrigate and it will work with solar panel
  • The system use soil moisture sensor, temperature sensor, and rain sensor to collect precise environmental data for real-time irrigation control.
  • In the future, I plan to enhance the system by integrating additional climate sensors and connected features for remote control via a mobile app.
